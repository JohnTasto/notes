# Terminology

\[
  \begin{matrix}
    y = ax^2 + bx + c \\[.5em]
    \begin{aligned}
              \text{terms: } & ax^2, bx, c \\
       \text{coefficients: } & a, b \\
    \end{aligned}
  \end{matrix}
\]

\[
  \begin{matrix}
    \sqrt[n]{x} \\[.5em]
    \begin{aligned}
         \text{index: } & n \\
      \text{radicand: } & x \\
    \end{aligned}
  \end{matrix}
\]
