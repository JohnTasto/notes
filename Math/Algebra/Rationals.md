# Rationals

## Rules

\[
  \begin{aligned}
    a^{-1} &= {1 \over a  }                                   \BG \\
    a^{-n} &= {1 \over a^n}                                   \BG \\[1em]
    \bigg({a \over b}\bigg)^{-1} &=       {b \over a}         \BG \\
    \bigg({a \over b}\bigg)^{-n} &= \bigg({b \over a}\bigg)^n \BG \\[1em]
    { 1          \over {a \over b}} &= { b \over a}           \BG \\
    { c          \over {a \over b}} &= {cb \over a }          \BG \\
    {{a \over b} \over  c         } &= {a  \over cb}          \BG \\
  \end{aligned}
\]
